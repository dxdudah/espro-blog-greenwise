# Espro Blog GreenWise

Um blog moderno e responsivo focado em conteúdo sobre sustentabilidade e práticas ecológicas.

## 📋 Sobre o Projeto

Espro Blog GreenWise é uma plataforma de blog desenvolvida para compartilhar conhecimento sobre sustentabilidade, práticas verdes e impacto ambiental.

## ✨ Características

- Design responsivo e moderno
- Interface amigável e intuitiva
- Conteúdo focado em temas ambientais
- Fácil navegação
- Otimizado para performance

## 🛠️ Tecnologias

- **CSS** - Estilização e layout
- **HTML5** - Estrutura semântica
- **JavaScript** - Interatividade (se aplicável)

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/dxdudah/espro-blog-greenwise.git
```

2. Navegue até o diretório do projeto:
```bash
cd espro-blog-greenwise
```

3. Abra o arquivo index.html no seu navegador
